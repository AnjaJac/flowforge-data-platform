# FlowForge Data Platform

FlowForge is a Dockerized data orchestration platform built to learn Apache Airflow, Docker, Polars, and modern Data Engineering practices.

## Objectives

- Learn Apache Airflow orchestration
- Learn Docker containerization
- Implement modular data pipelines
- Apply data quality validation
- Build automated testing workflows
- Implement CI/CD with GitHub Actions

## Architecture

Source → Raw → Staging → Core → Analytics

## Technology Stack

- Python 3.12
- Apache Airflow
- Docker & Docker Compose
- Polars
- PostgreSQL
- Pytest
- GitHub Actions
